今回は、モブを叩くとドラクエのようにダメージ量が一時的に表示されるmodです。
マインクラフトは、クライアントとサーバーで処理が異なります。
描画処理はクライアント側で行われますが、ダメージ量はサーバー側でしか処理されていません。
そのため、ダメージ量を描画するためには、サーバーからクライアントへダメージ量に関するデータを送る必要があります。
このコードでは、ダメージ量とダメージを受けたエンティティのIDからなるデータを管理するものをpopupフォルダに、サーバーからクライアントへ送る処理をnetworkフォルダにまとめています。
描画に関するイベントはクライアント側で処理されるので、clientフォルダに置いてあります。
モブのDamageに関するイベントはサーバー側が中心であるため、eventフォルダで管理しています。
Configには何も手を付けていないが、とりあえず残しています。
